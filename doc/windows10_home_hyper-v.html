Enable client side Hyper-V on Windows 10

  • Windows 10 Pro supports client side Hyper-V.
    • Enable client side Hyper-V from "Add-Remove Programs" --> "Turn Windows Features on or off".
    • Check the "Hyper-V" option as shown below.
    • Only this step is needed to enable Hyper-V on Windows 10 Pro.

  • Windows10 Home by default does not support client side Hyper-V.
    • Use the below script to enable Hyper-V on Windows Home.
    • This is required only on Windows 10 Home Edition.
    • pushd "%~dp0"
    dir /b %SystemRoot%\servicing\Packages\*Hyper-V*.mum >hyper-v.txt
    for /f %%i in ('findstr /i . hyper-v.txt 2^>nul') do dism /online /norestart /add-package:"%SystemRoot%\servicing\Packages\%%i"
    del hyper-v.txt
    Dism /online /enable-feature /featurename:Microsoft-Hyper-V-All /LimitAccess /ALL

  • Launch Hyper-V management console using virtmgmt from an admin command prompt.
