Setup 2 node Kubernetes cluster on Windows
In this article we'd set up a 2 node kubernetes cluster on Windows using Hyper-V.
  • Windows10 Pro comes with client side Hyper-V support.
    • Enable client side Hyper-V from "Add-Remove Programs" --> "Turn Windows Features on or off". Check this :

  • Windows10 Home by default does not support client side Hyper-V. Use the below script to enable Hyper-V on Windows Home.
    • pushd "%~dp0"
    dir /b %SystemRoot%\servicing\Packages\*Hyper-V*.mum >hyper-v.txt
    for /f %%i in ('findstr /i . hyper-v.txt 2^>nul') do dism /online /norestart /add-package:"%SystemRoot%\servicing\Packages\%%i"
    del hyper-v.txt
    Dism /online /enable-feature /featurename:Microsoft-Hyper-V-All /LimitAccess /ALL

  • Launch Hyper-V management console using virtmgmt from an admin command prompt.
